## Student Flow

### Week 1-2: Game Proposal
1. Review proposal.html (explore accordions, view examples)
2. Use proposal planner to outline thinking
3. Write proposal in Word/Google Docs
4. Submit through Canvas
5. Receive instructor feedback

### Week 2-3: Training Quest (Concurrent)
1. Begin Training Quest while waiting for proposal feedback
2. Work through 5 sections at own pace
3. Study professional review methodology
4. Take notes on review structure and style

### Week 3-4: Game Review
1. With proposal approved, use Training Quest notes
2. Write detailed game review
3. Submit outline + final review through Canvas
4. Receive feedback on both structure and content

## Troubleshooting

### Pages Look Plain
- Clear browser cache (Ctrl+Shift+Delete)
- Hard refresh (Ctrl+F5)
- Check CSS files loaded (F12 → Network tab)

### Interactions Don't Work
- Enable JavaScript (check browser settings)
- Check browser console for errors (F12)
- Try different browser to confirm
- Verify all files are deployed

### Mobile Layout Broken
- Check viewport meta tag is in head
- Test on actual device (Chrome DevTools mobile view works differently)
- Clear mobile browser cache

### Can't Submit Proposal
- Remember: Tool doesn't submit (Canvas does)
- Copy/paste planner notes to Canvas
- Or write in Word, then upload to Canvas
